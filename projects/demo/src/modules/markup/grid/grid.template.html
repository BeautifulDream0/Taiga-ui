Класс tui-container предназначен для формирования контейнера для контента.
Класс tui-row и подмножество классов tui-col_* предназначены для формирования сетки внутри контейнера.

Классы контейнера

  • .tui-container: ширина 1104px на экранах >= 1360px и 824px на экранах <= 1359px
  • .tui-container_adaptive: ширина 1104px на экранах >= 1360px, 824px на экранах >= 1024px и <= 1359px, 608px на экранах >= 768px и <= 1023px, 512px на экранах >= 600px и <= 767px, 100% на экранах < 600px
  • .tui-container_fullwidth: ширина 100% с отступами по краям в 24px на экранах >= 1024px и 16px на экранах <= 1023px
  • .tui-container_fixed: ширина 1104px

Колоночные классы для адаптивной сетки

  • .tui-col_lg-*: ширина экрана >= 1360px (Desktop L)
  • .tui-col_md-*: 1024px =< ширина экрана <= 1359px (Desktop M)
  • .tui-col_sm-*: 600px =< ширина экрана <= 1023px (Tablet)
  • .tui-col_s-*: 600px =< ширина экрана <= 767px (Tablet S)
  • .tui-col_xs-*: 320px =< ширина экрана <= 599px (Mobile)
