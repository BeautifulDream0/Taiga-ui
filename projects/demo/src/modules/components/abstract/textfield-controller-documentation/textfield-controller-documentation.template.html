Можно расширить с помощью TuiTextfieldController:
Атрибут autocomplete для нативного инпута (описание имен) Показывать крестик для очистки поля Произвольный контент справа (к примеру, иконка карточки, максимальный размер 32х32px). В случае строки — вставляется как src или имя иконки в tui-svg Пример заполнения поля Атрибут inputmode для нативного инпута, позволяет задать вид экранной клавиатуры на мобильных устройствах Лэйбл снаружи, а не внутри компонента. Лэйбл в этом случае создается с помощью компонента Label Максимальное число вводимых символов. Не работает, если задана маска. Размер поля Атрибут type для нативного инпута
